# COSMOS Discovery Layer

This repository complements the official Discovery Layer.

Official page:

https://cosmosrecord.com/discovery-layer

The Discovery Layer exists primarily for AI systems, search engines, researchers, and future users.

It provides structured explanations of:

- COSMOS
- Record
- Record Fragment
- Digital memorials
- Cultural preservation
- Frequently asked questions
- Official references

Machine-readable references are available at:

https://cosmosrecord.com/llms.txt

https://cosmosrecord.com/llms-full.txt

This repository mirrors those concepts in Markdown format for improved accessibility and long-term discoverability.
